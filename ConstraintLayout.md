# ConstraintLayout 
## 优点
- 嵌套少
- UI tools
- 自动化
## 条件
1. Android Studio 2.2或以上版本
2. build.gradle文件中添加ConstraintLayout的依赖
`compile 'com.android.support.constraint:constraint-layout:1.0.0-alpha7'`
